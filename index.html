郵便番号入力

使用方法

以下をhtmlに追加
<script src="https://mao2009.github.io/zipToAddress/zipToAddress.js"></script>
住所検索欄を以下のように配置する zipToAddressの引数にはセレクタを指定する
                        
<div>
    郵便番号: 
    <input type="text" name="zip1"></input>　-　
    <input type="text" name="zip2"></input>
</div>
<div>
    住所 :
    <input type="text" id="pref"></input>
    <input type="text" id="city"></input>
    <input type="text" id="street"></input>
</div>
<div>
    <button onclick='zipToAddress("[name=zip1]", "[name=zip2]", "#pref", "#city", "#street")'>住所検索</button>
</div>
郵便番号がひとつのフィールドに入っている場合は2番目の引数を""にする。
郵便番号はハイフンが入っていても全角数字でも可能
